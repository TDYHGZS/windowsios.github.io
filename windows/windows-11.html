安装指南

开始下载 ISO 之前

确保你：

  • 拥有 Internet 连接（可能需要缴纳 Internet 服务提供商费用）。
  • 在要下载 .iso 文件的计算机、USB 或外部驱动器上有足够的可用数据存储空间。
  • 至少具有 8GB（和 DVD 刻录机）的空白 DVD 光盘，用于创建可启动光盘。我们建议使用空白 USB 或 DVD，因为在安装过程中其中的任何内容都将被删除。
  • 如果在尝试从 ISO 文件刻录 DVD 可启动光盘时收到“光盘映像文件过大”消息，请考虑使用更高容量的双层 DVD。

安装说明 (DVD)

  • 找到你下载 Windows 11 ISO 的文件夹。
  • 若要使用 Windows 磁盘映像刻录机创建安装 DVD，请右键单击 ISO 文件，然后选择 属性 。在"常规选项卡"下，单击"更改..."，其中显示"打开方式："，然后选择“Windows 资源管理器”。选择 应用 。这将使 刻录光盘映像 能够在你右键单击 ISO 文件时显示。
  • 要使用第三方 DVD 刻录程序，你可以右键单击 ISO 文件，然后选择 打开方式 。

安装说明 (ISO)

  • 如果你要从 ISO 文件直接安装 Windows 11，而不使用 DVD 或 USB 闪存驱动器，则可以通过装载 ISO 文件来执行此操作。此时，会将你当前的操作系统升级到 Windows 11。
  • 若要装载 ISO 文件，请执行以下操作：
  • 转到下载 ISO 文件的位置。然后，右键单击 ISO 文件并选择 属性 。
  • 在"常规"选项卡上，单击"更改..."，其中显示“打开方式：
  • 选择 Windows 资源管理器作为用于打开 ISO 文件的程序，然后选择 应用 。
  • 右击 ISO 文件并选择 挂载 。
  • 这将创建一个“虚拟”可启动光盘。双击即可查看里面的文件。双击 setup.exe 开始 Windows 11 安装。
