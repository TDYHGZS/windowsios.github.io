工具包简介

Windows®自动安装工具包（Windows AIK）是一套用于配置和部署Windows®操作系统的工具与文档集合。通过Windows AIK，用户可实现以下功能：

  • 自动化Windows安装流程
  • 使用ImageX捕获Windows映像
  • 通过部署映像服务和管理（DISM）配置、修改映像
  • 创建Windows PE映像
  • 借助用户状态迁移工具（USMT）迁移用户配置文件及数据
  • 利用卷激活管理工具（VAMT）集中管理多次激活密钥（MAK），实现卷激活自动化

系统要求

支持的操作系统：

  • Windows 7 Service Pack 1
  • Windows Server 2008 R2 SP1
  • Windows Server 2003 Service Pack 2
  • Windows Vista SP1
  • Windows Server 2008系列
  • Windows 7系列
  • Windows Server 2008 R2系列
