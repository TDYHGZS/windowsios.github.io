下载 Windows 11 ISO

开始下载前的准备

确保您具备：

  • 拥有 Internet 连接（可能需要缴纳 Internet 服务提供商费用）。
  • 在要下载 .iso 文件的计算机、USB 或外部驱动器上有足够的可用数据存储空间。
  • 至少具有 8GB（和 DVD 刻录机）的空白 DVD 光盘，用于创建可启动光盘。我们建议使用空白 USB 或 DVD，因为在安装过程中其中的任何内容都将被删除。
  • 如果在尝试从 ISO 文件刻录 DVD 可启动光盘时收到"光盘映像文件过大"消息，请考虑使用更高容量的双层 DVD。

确保要安装 Windows 11 的 PC：

  • 拥有 64 位 CPU：Windows 11 只能在 64 位 CPU 上运行。若要查看你的 PC 是否有 64 位 CPU，请转到 设置 > 系统 > 关于 ，或在 Windows 中搜索"系统信息"，然后在"系统类型"下查看。
  • 满足系统需求：并非所有运行 Windows 10 的设备都有资格获得 Windows 11 升级。

警告：不建议在不符合 Windows 11 最低系统需求的 PC 上安装 Windows 11 媒体，这可能会导致兼容性问题。如果你继续在不符合要求的 PC 上安装 Windows 11，则该 PC 将不再受支持，并且无权接收更新。由于缺乏兼容性而导致的 PC 损坏不在制造商保修范围内。

下载完成后，你可以使用 验证下载 选项来确保文件的完整性。

注意：有关升级要求和支持的功能，请参阅 Windows 11 设备规格。某些功能需要额外的硬件。我们还建议你访问 PC 制造商的网站，了解有关更新的驱动程序和硬件兼容性的信息。

安装说明

安装说明 (DVD)：

  1. 找到你下载 Windows 11 ISO 的文件夹。
  2. 若要使用 Windows 磁盘映像刻录机创建安装 DVD，请右键单击 ISO 文件，然后选择 属性 。在"常规选项卡"下，单击"更改..."，其中显示"打开方式："，然后选择“Windows 资源管理器”。选择 应用 。这将使 刻录光盘映像 能够在你右键单击 ISO 文件时显示。
  3. 要使用第三方 DVD 刻录程序，你可以右键单击 ISO 文件，然后选择 打开方式 。

安装说明 (ISO)：

如果你要从 ISO 文件直接安装 Windows 11，而不使用 DVD 或 USB 闪存驱动器，则可以通过装载 ISO 文件来执行此操作。此时，会将你当前的操作系统升级到 Windows 11。

  1. 若要装载 ISO 文件，请执行以下操作：
  2. 转到下载 ISO 文件的位置。然后，右键单击 ISO 文件并选择 属性 。
  3. 在"常规"选项卡上，单击"更改..."，其中显示“打开方式："然后选择 Windows 资源管理器作为用于打开 ISO 文件的程序，然后选择 应用 。
  4. 右击 ISO 文件并选择 挂载 。
  5. 这将创建一个"虚拟"可启动光盘。双击即可查看里面的文件。双击 setup.exe 开始 Windows 11 安装。
